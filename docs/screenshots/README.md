# VirSift screenshot assets

The `previews/` directory contains lightweight README images.

The `full/` directory contains high-resolution screenshots or readable
multi-panel overviews. Select a README preview to open its full-resolution
counterpart.

Current sequence:

1. Landing
2. Surveillance Observatory
3. Workspace / activation
4. Sequence Refinery
5. Molecular Timeline
6. Analytics
7. Export
8. Documentation

`05-molecular-timeline.png` is intentionally a four-panel overview rather than
one extremely tall screenshot. This keeps the interface text readable when the
full image is opened.
